# Local LLM Skill Packs

This directory contains prompt packs / skill docs that can be loaded into a **local LLM client** without Hermes.

## Available

- [DSA Aventurica Research Skill](dsa-aventurica-research.md) — source-backed Wiki Aventurica research for adventure prep

## How to use

1. Open the skill markdown file.
2. Paste it into your local client’s custom instructions, system prompt, or pinned context.
3. Ask the model to follow the skill’s output format.
4. Keep Hermes out of the loop if you only want a local client workflow.

## Recommended local clients

- LM Studio
- Open WebUI
- any OpenAI-compatible local chat client
